# CodeAlpha_ChatBot
***Simple Python Chatbot***

A beginner-friendly chatbot built using Python. The chatbot interacts with users through the command line and provides responses to predefined 

questions using conditional statements.

***Features***

- Responds to greetings like "hello"
  
- Answers "how are you"
  
- Says goodbye when the user enters "bye"
  
- Handles unknown inputs with a default response
  
- Simple and easy-to-understand code

***Technologies Used***

- Python 3

***How to Run***

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/simple-python-chatbot.git
2.Run the program:

    python chatbot.py
